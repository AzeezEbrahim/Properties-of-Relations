# Properties-of-Relations
A (binary) relation R between sets X and Y is a subset of X×Y.  (X×Y is a Cartesian product.) 
It may have the following properties:
1- Reflexivity
2- Irreflexivity
3- Symmetry
4- Antisymmetry
5- Asymmetry
6- Transitivity
